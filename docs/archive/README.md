# Archivo

Esta carpeta conserva material historico que no debe usarse como documentacion
principal de entrega.

Contenido:

- `desarrollo/`: documentos de Parte A/B escritos durante el desarrollo.
- `laboratorio-20260702/`: notas del turno de laboratorio antes de cerrar Parte C.
- `decisiones/`: notas tecnicas historicas y decisiones tomadas durante el TP.

Algunos documentos de archivo mencionan pendientes o estados incompletos de Parte
C. La fuente de verdad para la entrega final es el README principal, el PDF final
y la documentacion fuera de `archive/`.
